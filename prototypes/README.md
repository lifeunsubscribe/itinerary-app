# Prototypes

Single-file predecessors of this app, from September 2025. Each is a
self-contained React 18 + Babel-standalone + Tailwind (CDN) page that runs
by opening it directly in a browser — no build step.

- `itinerary-app.html` — first prototype ("Travel Itinerary Planner")
- `itinerary-app-2.html` — second iteration, first titled "Wanderly"

The modular CRA rewrite in this repository grew out of these files. Kept
for project history; not maintained.
